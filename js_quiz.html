2. What does the map() method do? 
var array1 = [1, 4, 9, 16];

const map1 = array1.map(x => x * 2);

console.log(map1);

The map() method creates a new array with the results of calling a provided function on every element in the calling array.  // expected output: Array [2, 8, 18, 32]
